# What Does a #1 Hit Actually Sound Like?
### A Data Analysis of Billboard Hot 100 Chart History and Spotify Audio Features

A data analysis project combining Billboard chart history with Spotify audio 
features to explore the sonic characteristics of #1 hits from 1959 to 2019.

## Project Overview
This notebook analyzes 917 unique Billboard #1 hits to answer the question: 
is there a formula for a hit song? Using Python, pandas, SQLite, and 
Matplotlib/Seaborn, the analysis explores audio features including danceability, 
energy, valence, tempo, and acousticness.

## Tools Used
- Python, pandas, SQLite
- Matplotlib, Seaborn
- Scikit-learn (Euclidean distance calculation)
- Jupyter Notebook

## Data Sources
Data is not included in this repository due to file size. Download from:
- **Billboard Hot 100:** https://www.kaggle.com/datasets/ludmin/billboard
- **Spotify Hit Predictor:** https://www.kaggle.com/datasets/theoverman/the-spotify-hit-predictor-dataset

## To Run This Notebook
1. Download both datasets from the links above
2. Place all CSV files in the same folder as the notebook
3. Update the file paths in the data loading cell to match your local directory
4. Run all cells top to bottom
